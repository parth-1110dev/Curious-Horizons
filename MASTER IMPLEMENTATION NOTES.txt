MASTER IMPLEMENTATION NOTES

(Save this. This is now our implementation bible.)

1. Core Principle

This is NOT a redesign.

Never:

move components
resize sections
redesign layouts
redesign interactions
add/remove UI
change navigation
change functionality

Only implement the approved visual rebrand.

2. Visual Language

Every page must follow:

Deep cosmic navy / near-black backgrounds
Very subtle starfield
Soft atmospheric gradients
Gentle horizon glow
Restrained gold accents
Soft blue ambient shadows
Premium glassmorphism
Thin gold borders
Minimal blur
Large breathing whitespace

The feeling:

Apple

OpenAI

Interstellar

Never:

Cyberpunk

Gaming UI

Sci-fi dashboard

Neon

3. Color Palette

Primary Background

Ambient

Transparent Gold

4. Animations

Only use:

Very slow background drift

Tiny star movement

Soft ambient glow

2px hover elevation

Button glow

Subtle opacity transitions

Small blur fade

No flashy motion.

5. Glassmorphism

Cards:

Very subtle blur

Soft transparency

Gold border

Blue ambient shadow

No heavy frosted glass.

6. Typography

Large cinematic headings.

Generous spacing.

Readable line height.

No decorative fonts.

Premium.

Minimal.

7. Buttons

Primary:

Gold fill

Dark text

Soft glow

Rounded

Secondary:

Dark glass

Gold outline

Hover glow

No gradient buttons except the final CTA.

8. Hover Effects

Cards

2px lift

Gold border slightly brighter

Shadow slightly stronger

Buttons

Small glow

Tiny lift

Nothing excessive.

9. Session Reading Experience

Increase maximum reading width.

Current content container is too narrow.

Allow more comfortable long-form reading.

Do NOT change page layout.

Only increase readable width.

19. Overall Feeling

Every page should answer this question:

Ship it.

20. Performance Rule

Never sacrifice performance for visuals.

Animations should always remain GPU-friendly.

Avoid heavy particle systems.

Prefer CSS transforms and opacity over expensive effects.
